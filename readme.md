#What is it?
--------------

This is a game made with HTML5 canvas, CSS and native Javascript. It's loosely modeled after the popular Legend of Zelda series, complete with hero Link, villain Ganondorf, rupee currency and more.

##Languages Used
---------------
*HTML
*CSS
*Native Javascript
